This plugin will add a new field in the profile page to enable you to edit the URL of authors home page.

== Description ==

This plugin will add a new field in the profile page to enable you to edit the user_nicename field which controls the URL of authors home page.

Example of author's default home URL (with permalink):

http://www.example.com/blog/authors/author-default-name/

Can be changed to this:

http://www.example.com/blog/authors/author-new-name/

== Installation ==

To install this plugin, you need to do the following:

1. Unzip the downloaded file from the WordPress plugins directory.
1. Upload the unzipped "**author-slug-field**" directory to your WordPress plugin directory "wp-content/plugins".
1. Activate the plugin.
